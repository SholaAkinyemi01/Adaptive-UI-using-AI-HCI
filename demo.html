Adaptive UI Demo

Demonstrates an adaptive user interface using a simple multi-armed bandit and rule-based personalization. No server—everything runs in your browser.

Help us adapt this UI

With your permission, this demo stores anonymous interaction data in your browser (localStorage) to personalize layout, font sizes, and help prompts. You can revoke consent anytime.
